## 创作 FAQ

- `Q` 这是用什么画的？

  `A` 我在手机上，拿着手写笔，用自带的《三星笔记》app（与 Microsoft OneNote 相连）画的，用的就是笔刷工具，电子手绘。也许未来有一天，会用更专业的 app 来画。

- `Q` 图画的大小呢？

  `A` 这是数码绘画，尺寸比例与 A4 纸张类似，方向是竖着的，也就是横宽比例为根号2比1（√2:1）。

- `Q` 怎么设计出的这个形象？

  `A` 这很像我，就是这么设计出的这个形象。
