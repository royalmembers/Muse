# 朝鲜王朝

起床。又是艳阳高照的一天，刷了牙但不想洗脸，其实也不想起床，不过我亲爱的帅 Daddy 还是把我一把拉下了床，这个过程可为欢声笑语。

众所周知，我亲爱的帅 Daddy 是朝鲜国王，每天要在景福宫上班。没错，景福宫就是我的家，只不过我们一般住在后院；他在前殿上班，那个地方上面挂了一个牌子，写着勤政殿三个汉字。有时我也会去他那玩，但我的职务是朝鲜吉祥物，就是每天撒个娇再拍个照那种，不过大家最近总是说我吐舌头的照片不好看。

![景福宫勤政殿](./20250522_125522.webp)

我走出宫，在外面吃着哈根达斯，突然有一种奇妙的想法，那就是我想当大女王了，这可怎么办？看着不高不矮的宫墙，嘿嘿，我脑海中出现了一个好赞的计划。

![景福宫外侧景](./20250522_122925.webp)

我找来深色的迷魂药，于是爬在宫墙外，隔着方形的窗户上的方形洞孔，对着园内穿过华丽气派勤政殿的 Daddy，正准备往里吹，不过还没开始，我就自己舔了一口，啊真甜，我怎么可能会真的打算弄晕我亲爱的 Daddy 呢，其实这是一包巧克力粉，我开心地吃了起来。

![景福宫内一处宫殿内景](./20250522_131823.webp)

吃完后，我灵活地翻墙而入，啪嗒一声，莎莎莎，我快速向正往后花园走去的 Daddy 飞奔。宫内许多地方都铺设有白黄细纱，就是为了防止刺客潜入时能产生声响。

![景福宫后花园书房](./20250522_143226.webp)

突然，天上划过一个闪亮的流星，Daddy 身边武功高强全副武装的近内卫突然大叫一句护驾，然后急匆匆地围了上去，我在混乱和恍惚中也赶忙追了过去，大家一起往宫外北岳山中奔跑。此时我眼前似乎出现了涟漪，周围发生了变化，战乱和欢腾接二连三，就像看电影一样，等缓过神，发现在景福宫北门半山处一片平整的大草坪上，周围是蓝瓦木屋，随行人员现在都是西装革履，他们说这是总统官邸，并称呼我的 Daddy 为大统领，这是怎么回事呢？

![大统领官邸](./20250522_154401.webp)

![大统领官邸](./20250522_154546.webp)

我们往山下走去，有一座大型蓝色房顶的建筑物，听说叫做青瓦台，是大韩民国的总统府，是 Daddy 新的上班地方，我们走了进去，里面很大，有会议室和办公室，我突然跑去抱着正在处理事物的 Daddy，突然听到说该吃早饭了，可是这一天都过了这么久了怎么还吃早饭呢真搞笑，我闭着眼笑。

![青瓦台外景](./20250522_145639.webp)

![青瓦台总统办公室](./20250522_152653.webp)

猛地一睁眼：咦？我正躺在酒店套房柔软大床上抱着 Daddy，而他正对着我叫我快起床吃酒店自助呢。卧房内开着电视，新闻里说着韩语，不过我似乎听到了，主持人姐姐说清晨天未亮时突然出现了一个巨大漂亮的流星划过，而 Daddy 在和我说，他昨晚做了一个梦，梦见我们穿越到了古时候的朝鲜王朝…
